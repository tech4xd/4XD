# 4XD IT Solutions Website

A clean, organized static website for 4XD IT Solutions.

## Project Structure

```
4XD/
├── index.html          # Home page
├── about.html          # About Us page
├── talk-to-us.html     # Contact page
├── style.css           # Consolidated styles (495KB)
├── scripts.js          # Consolidated JavaScript (229KB)
├── css/                # Individual CSS files (43 files)
├── js/                 # Individual JavaScript files (19 files)
├── images/             # Images and graphics (2 files)
└── fonts/              # Web fonts (empty for now)
```

## Features

- ✅ Clean folder structure
- ✅ Consolidated CSS and JavaScript
- ✅ Working navigation between pages
- ✅ All resources organized properly
- ✅ No external dependencies (works offline)

## Usage

Simply open `index.html` in your web browser to view the website.
